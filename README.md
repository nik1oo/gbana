![gbana](img/title.png)

GBA emulator.

# Usage

1. Run emulator.

2. Drag and drop GBA BIOS.

3. Drag and drop game.

# Compiling from source

```
git clone https://github.com/nik1oo/gbana.git
cd gbana
make
```
